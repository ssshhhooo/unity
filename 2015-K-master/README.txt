・すでに配置してあるオブジェクトに関しては、サイズ・位置・判定など、微調整済みです。
・さらにオブジェクトの追加を行う予定です。

=======

現在、プロジェクト部分には、以下のものが置いてあります。

 Animation	- 各種アニメーション系
 Editor		- エディタ系を司るためのコード
 Fonts		- フォント(現在は適当に拾ってきたもの)
 Materials	- Unity内で作成した物体(現在は空)
 Prefabs	- 複数のアイテムをひとまとめにしたもの(現在テストが置いてあります)
 Scenes		- 現在の作成状態を保存しています。とりあえずこれを実行してみてください。
 Scripts	- 作成したメイン機能のソースコード(随時変更予定)
 Sounds		- ゲーム内SE,BGM(現在は空)
 Sprites	- 各種素材(主に画像)
 StandardAssets	- バーチャルパッドのソースコード。割と手を加えてます。過半数未使用。
 Textures	- テクスチャ系(現在は空)

=======

現在、ヒエラルキー部分には、以下のものが置いてあります。

 Player		- 主人公に関する設定
 Objects	- オブジェクト(主に家具類)置き場
 Wall		- 壁
  WallObjects	- オブジェクト(壁)置き場
  WallCollision	- 一部の壁の衝突判定用オブジェクト
 TEST		- テスト用キャラクター
 Main Camera	- カメラ(視点用)。主人公を追随します。
 Input		- 入力部分(操作用)
  KeyCanvas	- 十字キー用オブジェクト(見た目)
  MobileStick	- 実操作部分(バーチャルパッド + 十字キー)
 EventSystem	- 入力や衝突判定の対応
